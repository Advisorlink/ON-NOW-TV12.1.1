Launcher Admin

Dock tiles

Drag the ↑ ↓ arrows to reorder. Upload a tile image (the card art) and a wallpaper (the background painted behind the dock when this tile is focused). Both should be JPEG. Recommended: tile image 1280×720, wallpaper 1920×1080.

    Image & wallpaper uploads save instantly — this button only saves label / sub / target / accent edits.

    APK manifest

    Apps that the launcher offers to install. Add either an APK file (upload) or a remote URL.

    Broadcast notification

    A popup that every launcher device shows on screen. Each device dismisses with OK.
